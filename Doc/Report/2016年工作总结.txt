CnPack 开发组 2016 年工作总结

    CnPack 在 2016 年取得的主要成绩如下：

一、CnPack IDE 专家包连续发布 1.1.0、1.1.1 版本，增加对 RAD Studio 10.1 Berlin 的支持，代码高亮重构以全面支持代码中的 Unicode 字符。

    每日构建版下载地址：
    https://www.cnpack.org/downbuilds.php

二、CnPack 源码继续于 GitHub 上维护。地址：https://github.com/cnpack

三、CnVcl 新增部分单元，包括 Redis Client、素数计算、SHA2、网络格式定义等。

四、CnPack 公益基金的帐户继续公开接受用户捐助，目前支持支付宝与腾讯财付通两种捐助方式。

    链接：
    https://www.cnpack.org/foundation.php
    公开帐目：
    https://www.cnpack.org/showdetail.php?id=529

    2017 年 CnPack 开发组的计划如下：

一、CnWizards 继续增强功能、修复 Bug，增加对新版本 IDE 的支持。

二、CnWizards 将进一步加强国内外的宣传工作。希望能继续进一步扩大用户群。

三、继续扩充及移植 CnPack 组件包，补充和完善各类文档。

四、继续进行 CnPack 开发组的团队建设与规模发展。

CnPack Working Report in 2016

    Below are the achievements of CnPack in 2016:

1. CnPack IDE Wizards 1.1.0/1.1.1 published. Support RAD Studio 10.1 Berlin. Refactor Source Highlight to Support Unicode Source.

   Our nightly build version:
   https://www.cnpack.org/downbuilds.php

2. CnPack Source Stays at GitHub. https://github.com/cnpack

3. CnVcl Add Redis Client, Prime Number, SHA2 algorithm, Network Definitions Units.

4. CnPack Foundation accepts Donation at alipay.com and tenpay.com (Chinese). For English, please visit SourceForge:
   https://sourceforge.net/project/project_donations.php?group_id=110999
